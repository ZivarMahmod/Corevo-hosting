# Template-katalog — beslutslogg

Frisör-templates som granskas → väljs → ombrandas till Corevo-katalogen.

**Flöde:** `00-inbox/` (rådump, gitignorad) → Zivar svep-väljer → `01-kandidater/` → Cowork deployar/öppnar index, klickar runt, tar screenshots (`screenshots/`) → vi väljer ihop → `02-valda/` → brief till Code för katalogen.

**Ombrandnings-regler (Zivars beslut 2026-06-07):**
- Corevo-branding på namn/text — **färgerna BEHÅLLS** per template
- Samma bildset i alla templates → snabbt att skicka till kund
- ⚠️ Licens per källa ska kollas innan en template går till `02-valda/`
