Projektmappen för plattformen. Planering och kod hålls isär — samma metodik som Sadaqa Sweden.

## Var saker bor

| Mapp | Innehåll |
|---|---|
| `1-Planering/` | VAD ska byggas — moduler, idéer, research, beslut |
| `2-Byggplan/` | HUR det byggs — roadmap + numrerade goal-briefs |
| `3-Bakgrund-Research/` | Stödmaterial, omvärld, referenser |
| `4-Dokument-Underlag/` | Underlag, exporter, råmaterial Zivar lägger in |
| `5-Kod/` | Själva koden — eget revir, egen git |
| `Nörden/` | Min plats — där jag dokumenterar saker som rör mig. Zivar lägger in filer här. |

## Flödet
