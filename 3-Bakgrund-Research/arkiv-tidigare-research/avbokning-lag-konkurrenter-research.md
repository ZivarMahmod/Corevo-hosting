# Research: avbokning/återbetalning — svensk lag + konkurrenter (2026-06-06)

Underlag till `1-Planering/avbokning-aterbetalning-modell.md`. Källkollat, ej juridisk rådgivning.
Låt jurist granska standardvillkoren före lansering (särskilt 100%-behåll-alternativet).

## TL;DR
- Ångerrätt (14 dagar) gäller i praktiken INTE en förskottsbetald, tidsbokad frisörtid — undantag i
  distansavtalslagen 2 kap. 11 § (tjänst på bestämd dag/tid + påbörjad/fullgjord tjänst med samtycke).
  MEN: plikt att informera att ångerrätt saknas INNAN avtal ingås.
- Avbokningsavgift/behållen betalning lagligt OM (1) tydligt presenterad + accepterad FÖRE betalning,
  (2) skälig (kopplad till faktisk förlust). ARN underkänner avgifter utan bevisad kostnad.
- 19 juni 2026 "ångerknapp" ÄKTA: prop. 2025/26:84, i kraft 2026-06-19, gäller varor + tjänster online.
  Styr HUR ångerrätt utövas — skapar den inte. Relevant för presentkort/klippkort (har ångerrätt tills
  inbokade), inte rena bokade tider.
- Konkurrenter: ägaren sätter fönster; utanför = full refund, inom = ingen/partiell/forfeited deposit;
  reschedule erbjuds nästan alltid; goodwill = manuell. Hård gräns på antal reschedules saknas hos de
  stora → "reschedule en gång → manuellt" differentierar.

## 1. Ångerrätt på bokad frisörtid
Faller bort via 2 kap. 11 § distansavtalslagen (lag 2005:59): (a) tjänst på bestämd dag/tidsperiod
(samma kategori som konsert/match/inkvartering), (b) tjänst fullgjord/påbörjad med konsumentens
uttryckliga samtycke + medgivande att ångerrätten upphör. Plikt: informera att ångerrätt saknas före
avtal — annars svårt att åberopa undantaget. Konsumenttjänstlagen gäller ej behandling på person.
Källor: Konsumentverket (lagen om distansavtal), Lawline (2 kap. 11 §), Konsumentverket (informationskrav).

## 2. Avbokningsavgift / behållen förskottsbetalning
Lagligt. Krav: (1) villkor tydligt angivet + uttryckligen accepterat INNAN bokning/betalning;
(2) skäligt enligt avtalsvillkorslagen (1994:1512) — spegla faktisk förlust, annars oskäligt → jämkning.
ARN 2019-11341: avgift underkänd då företaget inte bevisade kostnaden. Logga samtycke (tidsstämpel +
villkorsversion). Moms på no-show/avbokningsavgift hanteras separat (Skatteverket) — fakturalogik.
Källor: Familjens Jurist, Lawline (uteblivet frisörbesök), Konsumentverket (avtalsvillkorslagen),
ARN 2019-11341.

## 3. Ångerknapp 2026-06-19 (prop. 2025/26:84)
Riksdagsbeslut 2026-03-11, i kraft 2026-06-19. Ångerfunktion vid avtal via onlinegränssnitt: lätt
tillgänglig + tydligt märkt under hela ångerfristen; konsument anger namn + avtalsuppgifter + hur
mottagningsbevis skickas; näringsidkaren skickar mottagningsbevis i varaktig elektronisk form. Gäller
varor OCH tjänster. Saknad knapp = otillbörlig marknadsföring. Nyans: styr utövande där ångerrätt finns
(t.ex. presentkort/klippkort innan inbokning), inte bokade tider utan ångerrätt.
Källor: Regeringen (pressmeddelande), Gulliksson, Setterwalls, Moll Wendén, Energiföretagen.

## 4. Konkurrenter (förskott, sen avbokning)
- **Bokadirekt (SE):** ägaren sätter fönster (t.ex. 24h); sen/no-show → betalning behålls; no-show-faktura.
- **Fresha:** ägaren sätter (ex. 48h); inom fönster avgift (t.ex. 25%); depositioner non-refundable men
  ägaren kan återbetala manuellt; betalning följer med vid reschedule.
- **Treatwell:** default 24h; utanför = full refund, inom = ingen refund; reschedule senast 1h innan.
- **Square Appointments:** ägaren sätter cut-off; ingen / full förskott / kort-hold; prepaid → kund kan
  ej självavboka, ägaren refunderar manuellt.
- **Acuity/Squarespace:** konfigurerbar avgift + deposition; reschedule kan stängas av; INGEN inbyggd
  gräns på antal reschedules (community efterfrågar).
- **Mindbody:** late-cancel + no-show-avgifter, ofta full kostnad.
- **SimplyBook.me:** mallar — gratis (full refund inom tid) / reschedule-only (ingen refund men gratis
  ombokning) / one-visit (full kostnad). Reschedule-only = anti-no-show-mönstret.

Mönster: ägaren väljer fönster; utanför = full refund, inom = ingen/partiell/forfeit; reschedule som
mjukare alternativ styrt av samma fönster; goodwill manuell; hård reschedule-gräns saknas hos de stora.

## 5. Rekommendation (försvarbar struktur)
1. Visa + kräv aktivt godkännande av avbokningsvillkoret FÖRE betalning (kryssruta, ej bara länk). Logga.
2. Informera tydligt att ångerrätt inte gäller den bokade tjänsten — före avtal.
3. 3 ägarval — riskgradering: 0%-behåll = högst risk (oskälighet, särskilt korta/billiga tider);
   partiell X% = lägst risk (default-rekommendation); 0% + reschedule = lägst friktion, starkast mot ARN.
4. "Reschedule en gång → manuell hantering": oproblematiskt + bra anti-loophole.
5. Bygg ångerknapp för produkttyper med ångerrätt (presentkort/klippkort) + mottagningsbevis; för rena
   tider: tydlig info att ångerrätt saknas.

Riskflaggor: 100%-behåll på korta/billiga tider; avgift utan förlust-koppling; ologgat samtycke; långa
fönster (72h) + 100%-behåll. Branschstandard 24-48h är säkrare.
